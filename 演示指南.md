# 🫘 豆角 — AI 多元化创作平台 · 演示指南

## 项目介绍

豆角是一个面向 AI 图片和视频生成的多元化创作平台，拥有 11 个页面、9 大功能模块。项目采用 Apple 极简暗色科技风格，全中文界面，配有 3D 吉祥物"豆角"贯穿全站，搭载丝滑动效。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | Next.js 16 + React 19 + TypeScript |
| 样式系统 | Tailwind CSS 4 + 自定义设计系统 |
| 动效引擎 | Framer Motion |
| AI 图片/视频 | Replicate (Flux / Wan 2.1) |
| AI 文本/对话 | DeepSeek |
| UI 组件 | Lucide Icons + shadcn/ui 风格 |
| 数据渲染 | React Markdown（表格/代码高亮） |

## 启动方式

打开终端，执行：

```bash
cd D:\豆角智能体
npm run start
```

浏览器访问 `http://localhost:3000`

## 演示亮点

1. **吉祥物动效** — 首页豆角 3D SVG 漂浮 + 眨眼 + 呼吸光晕
2. **粒子背景** — Canvas 60 个光点漂浮 + 连线
3. **毛玻璃材质** — 全站 glass morphism 风格
4. **AI 对话表格** — 聊天页面输入"生成产品对比表格"测试 Markdown 渲染
5. **图片生成流程** — 选择风格 → 输入 Prompt → 生成动画 → 结果展示
6. **响应式设计** — 手机/平板/桌面均可正常显示

## 环境变量

需在 `.env.local` 配置：

```env
REPLICATE_API_TOKEN=    # https://replicate.com 注册获取
DEEPSEEK_API_KEY=sk-xxx # https://platform.deepseek.com 注册获取
```

未配置时自动使用 Mock 演示数据，不影响页面展示。
